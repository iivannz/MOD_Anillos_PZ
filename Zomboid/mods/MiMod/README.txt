ESTE ES UN MOD DE PROJECT ZOMBOID

Si el juego no reconoce este mod, verifica:

1. La carpeta debe estar en: ~/Zomboid/mods/MiMod/
   O en la carpeta de mods del juego de Steam

2. El archivo mod.info debe estar en la raíz de la carpeta MiMod

3. La estructura debe ser:
   MiMod/
   ├── mod.info
   ├── preview.png (opcional)
   └── lua/
       ├── client/
       ├── server/
       └── shared/

4. Reinicia completamente el juego después de activar el mod

5. Verifica en el menú principal -> Mods que el mod aparezca
